什么是状态模式？

状态模式：让一个对象的行为随着内部转股固态的改变而变化，而该对象也是像换了类一样。

为什么要用状态模式？

在复杂的状态更换之下，减少if-else,或者switch的使用，达到简化逻辑的目的，并且很好的分离各种模块（副作用是会多出很多的类以及接口）

适合的情况

其实适合的情况还是蛮多的

  1. 游戏场景的转换
  2. 游戏角色状态的改变：比如war3中的经典角色，熊德，当他是人形的时候是一套技能，当他变身成熊的时候，又是一套技能，并且两者之间的血量，属性，均不相同。
  3. ...

ScenesController

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using SceneStateController;
using UnityEngine.SceneManagement;

public class ScenesController
{
    // Start is called before the first frame update
    private bool RunStart = false;
    private SceneStateBase NowScene;
   public void SetState(SceneStateBase NewScene,string NewScenName)
   {
        LoadScene(NewScenName);
        if(NewScene!=null)
        {
            NewScene.ScenesEnd();
        }
        NowScene = NewScene;
   }

    // Update is called once per frame
   void LoadScene(string SceneName)
   {
        RunStart = false;
        Debug.Log("LoadScene " + SceneName);
        if(SceneName.Length==0||SceneName==null)
        {
            //return;
        }
        Application.LoadLevel(SceneName);
   }
    void StateUpdate()
    {
        if(Application.isLoadingLevel)
        {
            return;
        }
        if(NowScene!=null&&RunStart)
        {
            NowScene.ScenesStart();
        }
    }
}

SceneStateBase

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

namespace SceneStateController{
    public class SceneStateBase
    {
        /// <summary>
        /// 场景实现接口类，场景载入必须实现以下函数
        /// </summary>
        public virtual void ScenesStart()
        {

        }
        /// <summary>
        /// 场景结束之后必须调用这个函数，即切换场景之后所要做的处理
        /// </summary>
        public virtual void ScenesEnd()
        {

        }
    }
}

简单的写了场景的基类，所有场景都必须继承与他，重写两个函数，用于场景的载入，和切换的控制。

SceneStart
using QxFramework.Core;
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
namespace SceneStateController {
    public class SceneStart :SceneStateBase
    {
        /// <summary>
        /// 开始的时候要做什么呢
        /// </summary>
        public override void ScenesStart()
        {
            UIManager.Instance.Open("Main_UI");
            base.ScenesStart();
        }
        /// <summary>
        /// 结束了该怎么收尾呢，清楚关于自己场景的东西。
        /// </summary>
        public override void ScenesEnd()
        {
            base.ScenesEnd();
        }
    }
}
